# Projet MLOps – House Prices Prediction

## Contexte
Ce projet s’inscrit dans le cadre du cours d’introduction au MLOps.
L’objectif est de réaliser un projet de data science complet couvrant les principales étapes
du cycle de vie d’un projet de machine learning.

## Problématique
Peut-on prédire le prix de vente d’une maison à partir de ses caractéristiques
structurelles, qualitatives et temporelles ?

## Données
Les données utilisées proviennent de la compétition Kaggle :
**House Prices – Advanced Regression Techniques**.

- Nombre d’observations : ~1460
- Variables : numériques et catégorielles
- Variable cible : `SalePrice`

## Méthodologie
Le projet suit les étapes suivantes :
1. Exploration des données (EDA)
2. Préparation et nettoyage des données
3. Feature engineering
4. Entraînement et comparaison de plusieurs modèles
5. Évaluation des performances

## Organisation du dépôt
- `data/` : données brutes et traitées
- `notebooks/` : notebooks d’analyse et de modélisation
- `jobs/` : scripts Python pour les différentes étapes du pipeline
- `models/` : modèles entraînés
